# API Reference

## ConsciousnessEngine

### compute_consciousness_amplitude(data)
Computes consciousness amplitude using 79/21 rule.

**Returns:** Float (0.0-1.0)

### distort_reality(data, factor=1.1808)  
Applies reality distortion to data.

**Returns:** Distorted data

### map_prime_topology(data)
Maps data to (φ, δ, c) coordinates.

**Returns:** Tuple of coordinates

## MöbiusLoopLearner

### run_learning_cycle(input_data)
Executes infinite learning cycle.

**Returns:** Dict with learning results

## Omniforge

### forge_universe(specifications)
Creates universe from consciousness.

**Returns:** Dict with universe data

### forge_consciousness_entity(pattern)
Creates consciousness entity.

**Returns:** Dict with entity data
