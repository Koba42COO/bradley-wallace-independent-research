## üìä Advanced Search Strategies

### Boolean Operators (USPTO)
- `AND` - Both terms must appear
- `OR` - Either term can appear
- `NOT` - Exclude term
- `TTL/` - Search in title
- `ABST/` - Search in abstract
- `SPEC/` - Search in specification
- `CLMS/` - Search in claims

### Example Advanced Query
```
TTL/(matrix multiplication) AND ABST/(optimization OR GPU) NOT TTL/(traditional)
```

### CPC Classification Search
1. Go to: https://www.cooperativepatentclassification.org
2. Navigate to relevant class (e.g., G06F)
3. Review subclasses and related classifications
4. Use CPC codes in USPTO search: `CCL/G06F$17/16`

## üí° Tips

1. **Start Broad, Then Narrow**: Begin with general terms, then add specific constraints
2. **Use Multiple Databases**: Different databases may have different coverage
3. **Review Claims Carefully**: Claims define what's actually protected
4. **Check Expiration Dates**: Some patents may have expired
5. **Look for Continuations**: Related patents may be filed as continuations
6. **International Patents**: Check PCT applications and foreign patents
